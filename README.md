# 📊 E-Commerce Sales Data Analysis

I completed this project as part of my Data Analyst assignment.
In this project, I worked with real-world sales data and performed end-to-end data analysis including cleaning, processing, visualization, and generating insights.

## 🎯 Problem Statement

The dataset contained messy and incomplete sales data.
My task was to clean the data, analyze it, and answer business-related questions like:

* Which products generate the most revenue?
* What are the sales trends over time?
* How does customer purchasing behavior look?
* 
## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 🔄 Project Workflow

### 1️⃣ Data Understanding

* Loaded dataset
* Checked structure and data types
* Identified missing values
* 
### 2️⃣ Data Preprocessing

* Filled missing discount values using median
* Converted date column to datetime format
* Removed duplicates
* Created new features like:

  * `month`
  * `revenue`
    
### 3️⃣ Exploratory Data Analysis (EDA)

* Summary statistics (mean, median, std)
* Total and average revenue
* Top 5 products by revenue
* Sales by category
* Monthly sales trends
* Correlation analysis

### 4️⃣ Data Visualization

* Histogram for price distribution
* Bar chart for category revenue
* Line chart for monthly trend
* Scatter plot for quantity vs revenue
